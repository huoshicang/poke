# Poke for React

页面布局
- 头部
- 内容
    - 总数信息
    - 选择按钮
    - 内容
- 底部

整体：居中布局

总数信息、选择按钮：靠左对齐、允许换行

内容：网格布局（一行6列）靠左对齐

底部：居中

## 初步想法
采用url传参，保证可以刷新页面后，数据不变

选中项设置：
1. 选择类型
   1. 用来渲染网页url  
   可以实现多选，判断当前按钮项是否被选中
2. 选择url
   1. 用来获取内容  
   存储多个选中项的url
3. 获取到的内容
   1. 用来渲染页面

## 组件
- Header
    - 渲染头部信息
- Content
    - Total
      - 渲染总数信息
    - Button
        - 渲染选择按钮
    - Info
      - 渲染内容
        - InfoCard
          - 内容卡片
- Footer
    - 渲染翻页按钮
